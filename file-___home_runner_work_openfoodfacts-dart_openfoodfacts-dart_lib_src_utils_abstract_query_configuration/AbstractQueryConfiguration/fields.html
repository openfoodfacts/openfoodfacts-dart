fields property

List<ProductField>? fields
getter/setter pair

Defines which ProductFields are retrieved.

All not mentioned fields will be null in Product. Not specifying anything returns all available data (except Knowledge Panels). It's highly recommended to populate this field. This results in faster response times and less data usage.

   ProductQueryConfiguration config = ProductQueryConfiguration(
    '5449000131805',
    fields: [ProductField.ALL],
  );
  ProductResultV3 product = await OpenFoodAPIClient.getProductV3(config);

  print(product.product?.productName); // Coca Cola Zero
  print(product.product?.brands); // Coca-Cola

  ProductQueryConfiguration config = ProductQueryConfiguration(
    '5449000131805',
    fields: [ProductField.BRANDS],
  );
  ProductResultV3 product = await OpenFoodAPIClient.getProductV3(config);

  print(product.product?.productName); // null
  print(product.product?.brands); // Coca-Cola
